## HTML

Füge diesen Code in den `<head>` oder `<body>` Abschnitt deiner Website ein.

``` html
<script data-host="https://digistats.de" data-dnt="false" src="https://digistats.de/js/script.js" id="ZwSg9rf6GA" async defer></script>
```

::: tip INFO
Für eine reibungslose Funktionalität, empfehlen wir dir den digistats Tracking-Code im `<head>`-Bereich deiner Website einzufügen.
:::

## WordPress

Wenn du digistats Analytics in deine WordPress-Website einbinden willst, gibt es zwei Wege: Mit einem Plugin oder manuell.

### Mit Plugin

1. Installiere das [Kopf- und Fußzeilen einfügen](https://de.wordpress.org/plugins/insert-headers-and-footers/) Plugin.
2. Navigiere links im WordPress-Menü zu "**Einstellungen**" und wähle "**Kopf- und Fußzeilen einfügen**".
3. Kopiere den digistats Tracking-Code und füge es in die Fußzeilenskripte ein:

``` html
<script data-host="https://digistats.de" data-dnt="false" src="https://digistats.de/js/script.js" id="ZwSg9rf6GA" async defer></script>
```

4. Klicke auf **Speichern**.

::: tip INFO
Bitte sei dir bewusst, dass einige WordPress-Plugins Caching-Mechanismen einsetzen, die möglicherweise die Arbeitsweise von digistats Analytics beeinträchtigen. Als Nutzer solltest du sicherstellen, dass der Tracking-Code von digistats Analytics ohne Zwischenspeicherung (Caching) ausgeführt wird, um eine korrekte Funktionsweise des Trackings zu gewährleisten.
:::

### Ohne Plugin (Manuell)

Für die manuelle Einbindung ohne ein Plugin in WordPress, bearbeite die Datei header.php, die du in deinem WordPress-Verzeichnis findest. Diese befindet sich üblicherweise direkt im Theme-Ordner unter "**wp-content/themes/dein-theme/**".

1. Füge den gleichen digistats Analytics Tracking-Code wie oben direkt vor dem schließenden `</head>`-Tag ein:

``` html
<script data-host="https://digistats.de" data-dnt="false" src="https://digistats.de/js/script.js" id="ZwSg9rf6GA" async defer></script>
```

2. Speichere die Änderungen.

Wenn du weitere Hilfe bei der Installation von digistats Analytics benötigen, wende dich bitte an hallo@digistats.de.

## Squarespace

Um digistats Analytics in deine Squarespace-Website einzubinden, stell sicher, dass du einen Business- oder Commerce-Plan hast. Dann folge diesen Schritten:

::: tip INFO
Nur für Websites mit einem Squarespace Business- oder Commerce-Plan verfügbar.
:::

1. Klicke auf "**Einstellungen**", dann auf "**Erweitert**" und anschließend auf "**Code-Injection**".
2. Füge in der Fußzeile diesen Code ein:

``` html
<script data-host="https://digistats.de" data-dnt="false" src="https://digistats.de/js/script.js" id="ZwSg9rf6GA" async defer></script>
```
3. Klicke oben auf der Seite auf **Speichern**.

Das war's! Dein Skript sollte jetzt auf deiner Squarespace-Website aktiv sein.

Weitere Details bei [Squarespace](https://support.squarespace.com/hc/en-us/articles/205815908-Using-Code-Injection)

Wenn du weitere Hilfe bei der Installation von digistats Analytics benötigen, wende dich bitte an hallo@digistats.de.

## Ghost

Für das Einbinden von digistats Analytics in deine Ghost-Website folge diesen einfachen Schritten.

1. Gehe zu deinen Ghost **Admin-Einstellungen**.
2. Klicke im Seitenleistenmenü auf "**Code-Injection**".
3. Füge den Tracking-Code in den Abschnitt "**Fuß**" ein:

``` html
<script data-host="https://digistats.de" data-dnt="false" src="https://digistats.de/js/script.js" id="ZwSg9rf6GA" async defer></script>
```

4. Klicke auf **Speichern**.

So einfach ist das! Jetzt sollte digistats Analytics auf deiner Ghost-Website laufen. Wenn du mehr Details brauchst, öffne die Ghost-Dokumentation in einem neuen Fenster.

Weitere Details bei [Ghost](https://ghost.org/faq/analytics/#how-to-set-up-simple-analytics-for-ghost)

Wenn du weitere Hilfe bei der Installation von digistats Analytics benötigen, wende dich bitte an hallo@digistats.de.

## Webflow

Für das Einbinden von digistats Analytics in deine Webflow-Website folge diesen einfachen Schritten.

::: tip INFO

2. Klicke anschließend im Tab-Menü auf „**Benutzerdefinierter Code**“.
3. Füge den folgenden Tracking-Code in den Abschnitt „**Fußzeilencode**“ ein:


4. Klicke auf **Änderungen speichern**.
5. Klicke auf „**Veröffentlichen**“, um die Änderungen zu veröffentlichen.

Weitere Details bei [Webflow](https://university.webflow.com/article/how-to-add-custom-head-and-body-code-to-a-webflow-site)

Wenn du weitere Hilfe bei der Installation von digistats Analytics benötigen, wende dich bitte an hallo@digistats.de.

## WIX

Für das Einbinden von digistats Analytics in deine WIX-Website folge diesen einfachen Schritten.

::: tip INFO
Nur für Websites mit einem kostenpflichtigen WIX-Plan verfügbar.
:::

1. Klicke im Dashboard deiner Website auf „**Einstellungen**“ .
2. Klicke unter „**Erweiterte Einstellungen**“ auf die Registerkarte „**Tracking & Analytics**“ .
3. Klicke auf + Neues Tool und wähle im Dropdown-Menü Benutzerdefiniert aus.
4. Füge den Code von unten ein:

``` html
<script data-host="https://digistats.de" data-dnt="false" src="https://digistats.de/js/script.js" id="ZwSg9rf6GA" async defer></script>
```

5. Wähle die entsprechende Domäne aus. Hinweis: Diese Option wird nur angezeigt, wenn du über mehrere Domains verfügst.
6. Gib einen Namen für deinen benutzerdefinierten Code ein. Beispiel: digistats Tracking-Code.
7. Code zu Seiten hinzufügen: Wähle „**Alle Seiten**“ und lade den Code einmal.
8. Platziere den Code in: Wähle „**Body – end**“ aus.
9. Klicke auf „**Übernehmen**“.

Weitere Details bei [WIX](https://support.wix.com/en/article/embedding-custom-code-to-your-site)

Wenn du weitere Hilfe bei der Installation von digistats Analytics benötigen, wende dich bitte an hallo@digistats.de.

## Überprüfen, ob Tracking funktioniert

Um zu überprüfen, ob das Tracking auf deiner Website funktioniert, kannst du folgende Schritte durchführen:

1. **Website besuchen:** Öffne deine Website in einem neuen Browserfenster oder Tab. Stelle sicher, dass du eine Seite aufrufst, auf der das Tracking aktiv sein sollte.
2. **Neue Seite öffnen:** Navigiere auf deiner Website zu einer anderen Seite, um einen weiteren Seitenaufruf zu generieren.
3. **Dashboard überprüfen:** Navigiere zum digistats Analytics Dashboard, wo du die Tracking-Informationen einsehen kannst. 
4. **Daten kontrollieren:** Überprüfe, ob die Seitenaufrufe und Besucherzahlen korrekt angezeigt werden. Normalerweise sollten die Daten in Echtzeit aktualisiert werden.
Wenn keine Daten angezeigt werden oder du denkst, dass die Integration nicht korrekt funktioniert hat, gehe bitte die Dokumentation erneut durch und stelle sicher, dass:

Du den richtigen Code verwendet hast.
Der Tracking-Code korrekt in die Website eingefügt wurde (im richtigen Bereich und ohne Tippfehler).
Es keine Konflikte mit anderen Skripten oder Einstellungen auf deiner Website gibt.

::: tip INFO
Es ist auch hilfreich, Browser-Konsolen auf Fehler zu überprüfen, die darauf hinweisen könnten, dass etwas nicht richtig läuft. Wenn du nach all diesen Schritten immer noch Probleme hast, könnte es sinnvoll sein, den Support des Tracking-Tools zu kontaktieren oder einen Fachmann zu konsultieren.
:::

Wenn du weitere Hilfe bei der Installation von digistats Analytics benötigen, wende dich bitte an hallo@digistats.de.
