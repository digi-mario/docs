### Anzeigen

```
https://digistats.de/api/v1/stats/{id}
```

<Note>
Die `{id}` der Website findest du in den Einstellungen der jeweiligen Website (z.B. Website ID: 321).
</Note>

#### Beispiel für die Anfragen

```javascript curl
curl --location --request GET 'https://digistats.de/api/v1/stats/{id}?name={name}&from={from}&to={to}' \
--header 'Accept: application/json' \
--header 'Authorization: Bearer {api_key}'
```

| Parameter       | Typ          | Beschreibung  |
| :------------- |:-------------| :-------------|
| `name`      | `erforderlich` `string` | Der Name der Statistik. Mögliche Werte sind: `browser`, `campaign`, `city`, `continent`, `country`, `device`, `event`, `landing_page`, `language`, `os`, `page`, `pageviews`, `pageviews_hours`, `referrer`, `resolution`, `visitors`, `visitors_hours`, `session_time`.
 |
 | `from`      | `erforderlich` `string` | Das Startdatum `Y-m-d` Format.
 |
 | `to`      | `erforderlich` `string` | Das Enddatum im `Y-m-d` Format.
 |
 | `search`      | `optional` `string` | Die Suchanfrage.
 |
 | `search_by`      | `optional` `string` | Suche nach. Mögliche Werte sind: `value` für Wert. Standardwert: `url`.
 |
| `sort_by`      | `optional` `string` | Sortieren nach. Mögliche Werte sind: `count` für Count, `value` für Wert. Standardwert: `count`.
 |
 | `sort`      | `optional` `string` | Sortieren. Mögliche Werte sind: `desc` für Absteigend sortieren, `asc` für Aufsteigend sortieren. Standardwert: `desc`.
 |
  | `per_page`      | `optional` `int` | Ergebnisse pro Seite. Mögliche Werte sind: `10`, `25`, `50`, `100`. Standardwert: `10`.
 |
