### Anzeigen

```
https://digistats.de/api/v1/account
```

#### Beispiel für die Anfragen

```javascript curl
curl --location --request GET 'https://digistats.de/api/v1/account' \
--header 'Accept: application/json' \
--header 'Authorization: Bearer {api_key}'
```
